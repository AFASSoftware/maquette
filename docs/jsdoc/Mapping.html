Interface: Mapping

Mapping

Keeps an array of result objects synchronized with an array of source objects. Mapping provides a Mapping#map function that updates the Mapping#results. The Mapping#map function can be called multiple times and the results will get created, removed and updated accordingly. A Mapping can be used to keep an array of components (objects with a renderMaquette method) synchronized with an array of data. Instances of Mapping can be created using module:maquette.createMapping.

Members

results :Array.<Object>

The array of results. These results will be synchronized with the latest array of sources that were provided using Mapping#map.
