Interface Component

A component is a pattern with which you can split up your web application into self-contained parts.

A component may contain other components. This can be achieved by calling the subcomponents renderMaquette functions during the renderMaquette function and by using the resulting VNodes in the return value.

This interface is not used anywhere in the maquette sourcecode, but this is a widely used pattern.
