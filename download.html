You can download maquette using one of the following methods:

CDN

Add the following snippet to the <head> section of your html document: 

<script src="//cdnjs.cloudflare.com/ajax/libs/maquette/{{ site.maquette_version }}/maquette.min.js"></script>

Bower

Run the following command:

bower install maquette

NPM

Run the following command:

npm install maquette
