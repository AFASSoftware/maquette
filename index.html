Maquette

Minimalistic Virtual DOM library with support for animated transitions

Animated transitions

Maquette enables you to use animations when making changes to the user interface. All you have to do is render the elements that should currently be visible. Maquette will then run the animations that you specified to make the neccessary adjustments.

Fast

Maquette was built with performance in mind. This has resulted in an implementation that can easily render and diff thousands of nodes within a single frame.

Ultra lightweight

Maquette is a pure and simple library which is only 3Kb gzipped. Compare this to React (36Kb) or JQuery (29Kb).

Why Maquette

Anyone who ever created a complex web application user interface knows it is hard to keep it synchronized with underlying data. In web applications this involves creating the right DOM tree representing the data and then updating the DOM whenever the data changes.

Maquette uses a technique called virtual DOM to automatically update the DOM for you. All you have to do is provide a function which renders the desired 'virtual' representation of the DOM using the current data. Maquette will take care of updating the real DOM, using animated transitions if desired.
