Maquette

Minimalistic Virtual DOM library with support for animated transitions


Animated transitions

Maquette can use animations to make changes to the user interface. All you have to do is specify the desired animation for entering and exiting and render only visible elements. Maquette takes care of the rest.

Fast

Maquette was built up from the ground with performance in mind. This has resulted in an implementation that can easily render and diff thousands of nodes within a single frame.

Ultra lightweight

Maquette is a pure and simple library which is only 2.4Kb gzipped. Compare this to React (36Kb) or JQuery (29Kb).

Why virtual DOM

Anyone who ever created a complex user interface knows it is hard to keep it synchronized with underlying data. In web applications this involves creating the right DOM tree representing the data and then updating the DOM whenever the data changes.

Virtual DOM is a technique which automatically updates the DOM for you. All you have to do is provide a function which renders the desired 'virtual' DOM representing the current data. Maquette will update the real DOM, using animated transitions if desired.
