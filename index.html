Maquette

Minimalistic Virtual DOM implementation with support for animated transitions


Animated transitions

Maquette can use animations to make changes to the user interface. All you have to do is specify the desired animation for entering and exiting and render only visible elements. Maquette takes care of the rest.

Fast

Maquette was built up from the ground with performance in mind. This has resulted in an implementation that can easily render and diff thousands of nodes within a single frame.

Ultra lightweight

Maquette is a pure and simple library which is only 2.4Kb gzipped. Compare this to React (36Kb) or JQuery (29Kb).

Why

Anyone who has ever created complex user interfaces knows how hard it is to keep it in sync with the underlaying data. This usually involves creating the DOM tree to match the data and updating the DOM when the data changes. Virtual DOM is a technique that handles updating the DOM for you. All you have to do is provide a function which renders the virtual DOM based on the current data and maquette will update the real DOM, using animated transitions if needed.
