Tutorial

In this tutorial we will make you familliar with the virtual DOM technique and the maquette library. The application we are building in this tutorial is a simulator for css transitions. More specifically, we are going to make a remote control for a flying saucer. This tutorial is made for people who are familiar with HTML and Javascript.

Get started

In this first exercise you will create the flying saucer we are going to control. Thanks to some alien piece of CSS, the flying saucer only needs the following markup: <div class="saucer"> In maquette, the browser DOM is not created from HTML however, but from a notation called 'virtual hyperscript'. It is different from HTML in two important ways.

  • It is a template language, which means it combines data with markup.
  • It is plain Javascript, which means it runs very fast.

The `h()` function in virtual hyperscript is responsible for creating the DOM. The first parameter of this function is a CSS selector string, like "div.landscape", which results in a <div class="landscape">. The last parameter is an array of contents, text strings and other nodes.
