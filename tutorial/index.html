Tutorial

In maquette, the browser DOM is not created from HTML, but from 'virtual hyperscript'. It is different from HTML in two important ways.

  • It is dynamic, which means it can combine data with markup.
  • It is a plain Javascript function, which means it executes very fast and it can be debugged.

In this tutorial, we are going to build a simulator for css transitions. In other words, we are going to make a red square perform circus acts. Your first assignment is to create the flying saucer. It must be constructed using the following markup: <div class="saucer">Flying saucer</div>
